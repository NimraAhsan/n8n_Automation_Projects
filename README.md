1. Age Verification Workflow
Ye mera pehla automation workflow hai jo user ki age check karta hai.

Kaise kaam karta hai:

Manual Trigger: Workflow ko start karta hai.

Set Data: Fake user data (Name, Age, Status) generate karta hai.

If Node (Check Age): Check karta hai ke age 18 se zyada hai ya nahi.

Logic:

Agar age > 18 ho toh "User is adult" ka message aata hai.

Agar age < 18 ho toh "User is under 18" ka message aata hai.
